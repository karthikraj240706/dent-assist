# 🦷 DentAssist – The Complete Dental Student Companion

DentAssist is a premium educational web platform developed for Bachelor of Dental Surgery (BDS) students. It brings together essential clinical learning resources, instrument references, patient education, quizzes, appointment simulation, and productivity tools into a single modern web application.

Designed using only HTML, CSS, and Vanilla JavaScript, DentAssist demonstrates responsive web design, clean UI/UX principles, and practical educational features commonly used in dental colleges and clinics.
